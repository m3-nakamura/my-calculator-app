# 電卓アプリ 検証手順 (Walkthrough)

## 実装された変更内容
以下の3つのファイルを作成し、プレミアムなデザイン（グラスモーフィズム）を持つWeb電卓アプリを実装しました。

- `calculator/index.html`: アプリの構造
- `calculator/style.css`: プレミアムなスタイリング（ぼかし効果、グラデーション、アニメーション）
- `calculator/script.js`: 計算ロジック（四則演算、クリア、削除）

## 検証手順

### 手動テスト
Webブラウザで `calculator/index.html` を開き、以下の動作を確認してください。

1. **基本計算**:
   - `5` `+` `3` `=` -> **8** と表示されるか
   - `10` `-` `4` `=` -> **6** と表示されるか
   - `6` `×` `7` `=` -> **42** と表示されるか
   - `20` `÷` `4` `=` -> **5** と表示されるか

2. **機能ボタン**:
   - 数字を入力後 `DEL` を押して、最後の1文字が消えるか
   - 数字を入力後 `AC` を押して、全てクリアされるか

3. **特殊ケース**:
   - `0` で割り算をした場合、アラートが出るか、または適切な挙動をするか（現在はアラート実装）
   - 小数点計算 `1.5` `+` `1.5` `=` **3** となるか

4. **レスポンシブデザイン**:
   - ウィンドウサイズを小さくしても、計算機が中央に配置され、レイアウトが崩れないか

## 結果
- [ ] 全ての計算が正しく行われる
- [ ] UIが期待通りに美しく表示される
